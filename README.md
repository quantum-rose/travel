### 介绍：

- 包含首页和城市选择页，首页采用瀑布流 + 无限滚动
- 拥抱全新前端开发技术栈
- 针对 VS Code 深度定制，开发体验 UP! UP! UP!

### 技术栈

`pnpm`、 `Vite`、 `Vue3`、 `TypeScript`、 `Vue Router`、 `Pinia`、 `Vant4`、 `Scss`

> 不由得感慨，前端的变化真是快啊！
